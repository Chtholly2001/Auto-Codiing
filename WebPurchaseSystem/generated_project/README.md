# 用户登录注册系统

这是一个简单的用户登录注册系统，使用 Flask 框架开发。

## 功能特性
- 用户注册
- 用户登录
- 会话管理
- 简单的用户界面

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行应用
```bash
python app.py
```

或者使用启动脚本：
```bash
chmod +x run.sh
./run.sh
```

## 使用说明
1. 访问 http://localhost:5000
2. 点击"注册"创建新账户
3. 点击"登录"使用已有账户登录
4. 登录成功后可以查看欢迎页面
